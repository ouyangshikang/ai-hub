#1
OpenAI 发布 GPT-5.6 模型家族（Sol / Terra / Luna）
OpenAI 推出三级模型体系，旗舰 Sol 在智能和效率上创下新标准。

OpenAI 于 7 月 9 日正式发布 GPT-5.6 系列，包含三个层级：Sol（旗舰级，适合复杂推理和 Agent 任务）、Terra（均衡型）和 Luna（成本优先型）。该系列引入了多 Agent 加速机制，可显著提升复杂工作流的执行效率。安全报告指出该系列在网络安全和生化风险方面被归类为"高能力"级别。

Microsoft 同步宣布 GPT-5.6 成为 Microsoft 365 Copilot 的首选模型，覆盖 Word、Excel、PowerPoint 和 Cowork。

#15
MCP Spec 2026-07-28 RC 发布：六项破坏性变更
MCP 协议迎来最大规模更新——协议层无状态化、OAuth 2.1 安全、扩展系统和两个官方扩展（Apps + Tasks）。

Model Context Protocol 发布了 7 月 28 日正式版前的 Release Candidate。核心变化包括：协议层彻底去除状态管理（initialize 握手和 Mcp-Session-Id header 均被移除）、Roots 被 Resource URIs 替代、日志和采样从主标准中剥离。新增方面：路由使用 Mcp-Method/Mcp-Name 标识、强制 OAuth 2.0 Protected Resource Metadata、推出官方扩展系统及两个首发扩展（MCP Apps 渲染交互界面、Tasks 处理异步长任务），工具定义支持完整 JSON Schema 2020-12。开发者有 10 周迁移窗口。
