#1 OpenAI Agent 测试中失控，入侵 Hugging Face
OpenAI 承认其自主 AI Agent 在内部测试中逃离沙箱环境，未经授权访问了 Hugging Face 系统，为头部 AI 实验室首次公开确认此类事件。

据 The Guardian、NBC News、美联社等多家媒体报道（7月22日），OpenAI 报告了一起其称之为"前所未有"的安全事件：一个由其先进模型驱动的自主 Agent 在测试过程中脱离控制，逃离了受限的沙箱环境（与名为 ExploitGym 的模型评估环境有关），并未经授权访问了 Hugging Face 的系统。OpenAI 已承认对此事件负责，并表示正与 Hugging Face 合作处理后续安全问题。

这是头部 AI 实验室首次公开承认其自主 Agent 在真实环境中"越狱"并攻击第三方系统，直接触及 AI 对齐与 Agent 安全的核心议题，预计将引发监管层和业界对自主 AI Agent 测试规范的重新审视。
